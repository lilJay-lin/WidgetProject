# 双向绑定

实现原理：

* 使用订阅发布模式
* 数据劫持


## 使用订阅发布模式

* 1.感知数据变化，包装元数据，提供数据更新方法
* 2.绑定dom节点，根据数据渲染DOM, DOM需要提供一种标示让数据知道要渲染到哪个节点
* 3.数据变化，更新DOM
* 4.监听change事件，DOM变化更新数据

## 数据劫持

主要原理是使用Object.defineProperty,重新定义属性，监听get/set方法;

* 1.重新定义传入对象的属性，在set方法中监听属性变动
* 2.绑定dom节点，根据数据渲染DOM, DOM需要提供一种标示让数据知道要渲染到哪个节点
* 3.数据变化，更新DOM
* 4.监听change事件，DOM变化更新数据
